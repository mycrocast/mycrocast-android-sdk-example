Chat

interface Chat : Observable<T>

Interface for controlling the chat workflow of the mycrocast sdk.

This is the entry point for anything related to chat activity.

To receive updates for a specific chatroom, you first need to register and implement the observer and then join the specific chat with the provided functions. Changes within the chat are propagated though the observer
